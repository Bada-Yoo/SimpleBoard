🔍 Git 전략 및 협업 방식

- 브랜치 전략: `feature/기능명` 사용
- PR 사용 여부: ✅ 사용
- 커밋 메시지: 명확히 기능 중심으로 작성 (예: `feat: 게시글 작성 기능 구현`)
- 충돌 해결 경험: 이슈 발생 시 팀원간 회의와 소통으로 해결, 다른 브랜치에서 main으로 병합 시 충돌 발생하여 팀장이 수동으로 해결
